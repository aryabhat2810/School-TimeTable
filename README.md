# School Timetable

This project is a simple **School Timetable** designed using **HTML and CSS**.  
It displays the weekly class schedule in a structured and visually clear table format.

## Features
- Clean and well-structured timetable layout
- Break column highlighted using a different color
- Off period highlighted with grey background
- Uses HTML table elements like `rowspan` and `colspan`
- Simple and easy-to-understand design

## Technologies Used
- HTML5
- CSS3

## Purpose
This project is created for learning and academic purposes to understand:
- HTML tables
- CSS styling
- GitHub repository and GitHub Pages hosting
